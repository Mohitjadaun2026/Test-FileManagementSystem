# Backend Documentation

## Scope

Covers the Java backend implementation under `backend/`:

- `api` module (controllers/security/config/exception handling)
- `service` module (core business logic and batch processing)
- `dao` module (repositories/specifications)
- `model` module (entities + DTOs)

## Runtime Summary

- Spring Boot 3.3.x
- Java 21
- MySQL datasource
- OAuth2 client + JWT auth
- Spring Batch for async file processing
